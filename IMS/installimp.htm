Installation of each Volume !
Each Volume needs to be installed onto your computer….The whole installation process is fully automated….

There can be 2 situations :

First time installation of Islam Media Player
Installing any volume thereafter …
The Process of installation is as follows:

When you insert your Islam Media Player CD into the CDROM then CD will autorun, this cd will then check to find any existence of Islam Media Player on your hard drive.

If it does not find Islam Media Player software on your hard drive, the cd will prompt you to install it and the installation process is very simple…. Or if the cd does find any existence of Islam Media Player software on your hard drive, the upgrade process will take place for that particular volume…. A Restart of Islam Media Player will be required for the upgrade to be complete……

In Islam Media Player you should be able to view the new volume details…
